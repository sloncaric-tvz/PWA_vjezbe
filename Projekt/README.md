# Projekt za PWA

Pregled vijesti po kategorijama, registracija, prijava, izrada novih članaka i kategorija

## Upute

1. Kopiraj projekt u XAMPP `htdocs` direktorij.
2. Pokreni APACHE web server i MySQL servis.
3. U bazi podataka pokreni SQL skriptu iz projekta. KORISTI SAMO JEDNU
```text
novine.sql - automatski radi bazu i tablice

ILI

novine_tablice.sql - potrebno je ručno napraviti bazu 'novine' pa u njoj pokrenuti skriptu da se naprave i popune tablice
```
